# Season + region rules

The skill uses these to interpret context like "summer" or "Northeast in March" and match it to garment weight, fabric, and dress-code expectations.

> **Pair with `references/style-reasoning.md`** — that file holds the climate matrix (dry vs humid heat), the formality scale, and the classic-vs-trend filter. This file is the seasonal/regional inputs; the reasoning file is how to combine them into a *why* sentence per pick.

## Seasons (Northern Hemisphere)

| Season | Months | Garment cues |
|---|---|---|
| Winter | Dec–Feb | Heavy knits, flannel, wool trousers, outerwear (peacoat, waxed jacket), boots |
| Spring | Mar–May | Lightweight oxford, chinos, denim, light layers (sport coat over tee), suede chukkas |
| Summer | Jun–Aug | Linen, short-sleeve, lightweight tees, shorts, espadrilles, lighter colors |
| Fall | Sep–Nov | Flannel, henley, denim, sport coats, suede boots, layering |

For Southern Hemisphere requests, invert.

## Regional adjustment

Customer's event location modifies the season's defaults. Note the **heat type** column — that determines fabric choice (see climate matrix in `style-reasoning.md`).

### US regions (primary)

| Region | Heat type | Adjustment |
|---|---|---|
| Pacific Northwest (Seattle, Portland) | coastal mild → cold wet | Always rain-resistant outer; year-round layering |
| Northern California coast (SF, Marin, Sonoma) | coastal mild | Mild year-round; bring a layer even in summer (fog) |
| Southern California (LA, San Diego) | dry heat | Skew one season lighter than calendar suggests; coastal evenings cool |
| Desert Southwest (Phoenix, Las Vegas, Joshua Tree) | dry heat (extreme) | Linen/lightweight only; sun-protective long sleeves OK because they don't trap moisture |
| Mountain West (Denver, SLC, Aspen) | high-altitude / cold dry | Layering is mandatory; sun + cold combo; UV is high even when it's cold |
| Northeast (NYC, Boston) | humid heat (summer) → cold wet (winter) | Calendar-true; humid summers; cold transitional fall/spring |
| Mid-Atlantic & South (DC, Atlanta, Charleston) | humid heat | Summer = sticky; lightweight cotton/linen blends, NOT pure cotton oxford |
| Texas (Austin, Dallas, Houston) | humid heat (Houston) / dry heat (W. Texas) | Summer extreme; mild winter; spring/fall short |
| Florida (Miami, Tampa) | humid heat | Summer-like 8 months; rare cold snaps; avoid heavy cotton |
| Midwest (Chicago, Minneapolis) | humid heat (summer) → cold dry (winter) | Winters severe; humid summers |

If the customer specifies altitude (mountain wedding, ski trip, etc.) bias toward layering and warmer weights regardless of season. UV at altitude is intense even when ambient is cold — long sleeves are a feature, not a punishment.

## Dress codes (event-driven)

Match the explicit dress code if given. Otherwise infer from event type:

| Event | Default dress code | Buck Mason staples |
|---|---|---|
| Wedding (outdoor, daytime) | Smart casual / cocktail | Sport coat, dress shirt, trouser, dress shoe or chukka |
| Wedding (formal/black-tie) | Formal | Buck Mason isn't black-tie native — recommend pairing a dress shirt + black tie + outsourced tux/suit |
| Wedding (ceremony in barn/vineyard) | Smart casual | Sport coat, oxford or denim shirt, chino, suede dress shoe |
| Business meeting (East Coast) | Business casual / business | Wool trouser, dress shirt, blazer |
| Business meeting (West Coast tech) | Smart casual | Quality tee, button-down, denim, sport coat optional |
| Dinner reservation (upscale) | Smart casual | Dress shirt, trouser or dark denim, leather shoe |
| Travel day | Comfort smart-casual | Lightweight knit, denim, sneaker |
| Concert / casual social | Casual | Tee, henley, denim, jacket |

## Putting it together

Example: "wedding in Sonoma in late May, outdoor ceremony then dinner under string lights"

- Season: late spring → light layering, mild evenings.
- Region: NorCal coast/inland → fog-prone; bring a sport coat even if it's warm at noon.
- Dress code: smart casual → sport coat + dress shirt + trouser + suede chukka or dress shoe.
- Color palette: spring → lighter mid-tones (olive, stone, tobacco, indigo). Avoid heavy black or stark white.

The agent should write this reasoning out (briefly) when proposing an outfit, so the customer can correct any wrong assumption in one sentence.
